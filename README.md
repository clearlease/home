# Clearlea.se Website

This repository contains the source code for the official Clearlea.se website. Clearlea.se is an AI-powered platform designed for intelligent lease administration, automating the extraction and structuring of real estate documents.

## About This Project

This is a public repository that contains the code for the Clearlea.se marketing and information website. While the code is public, it is intended for internal use by the Clearlea.se team.

### Key Features of the Website

*   **Single-Page Application:** A modern, responsive single-page layout for a seamless user experience.
*   **Service Information:** Detailed sections on the challenges of manual data entry, the Clearlea.se workflow, use cases, and security measures.
*   **Interactive Elements:** The site uses Alpine.js for interactive components like the FAQ accordion and image carousel.
*   **Animations:** Lottie animations are used to visually explain the product's features.
*   **Contact Form:** A demo request form with reCAPTCHA integration.

## Tech Stack

The website is built with a simple and effective stack:

*   **HTML5:** For the core structure and content.
*   **Tailwind CSS:** For a utility-first CSS workflow.
*   **Alpine.js:** For lightweight JavaScript interactivity.
*   **Lucide Icons:** For clean and consistent icons.
*   **Lottie:** For high-quality web animations.

## File Structure

The repository is organized as follows:

```
.
├── assets/
│   ├── script.js       # Custom JavaScript for animations and form handling
│   └── styles.css      # Custom stylesheets
├── resources/
│   ├── content/        # Markdown files for legal pages (e.g., terms, privacy, imprint)
│   ├── favicon/        # Favicon assets for various platforms
│   ├── images/         # Marketing and illustrative images
│   ├── logos/          # Partner and company logos
│   └── lottie/         # Lottie animation JSON files
├── 404.html            # Custom 404 error page
├── terms.html          # General Terms and Conditions page (renamed from agb.html)
├── privacy.html        # Privacy Policy page (renamed from datenschutz.html)
├── impressum.html      # Imprint page
├── index.html          # The main landing page of the website
├── robots.txt          # Directives for web crawlers
├── sitemap.xml         # XML sitemap for search engines
└── README.md           # This file
```

## Getting Started

To view the website locally, simply open the `index.html` file in your web browser. As it's a static site, no server-side setup is required.
